# Market Breadth — TPTraders

Daily macro sentiment report covering the instruments that set the session tone.

**Live:** https://cbell0920.github.io/market-breadth  
**Archive:** https://cbell0920.github.io/market-breadth/archive.html  
**Hub:** https://cbell0920.github.io/tptraders-hub

## Covers
- Index futures: NQ, ES, YM, RTY
- Volatility: VIX spot + term structure
- Dollar: DXY
- Commodities: Gold, WTI Crude, Silver, Copper
- Yields: 2Y, 10Y, 30Y + curve spread
- Risk proxy: Bitcoin
- Sentiment: Put/Call ratio, Fear & Greed, NYSE A/D
- Session tone verdict: RISK-ON / RISK-OFF / NEUTRAL / MIXED

## Auto-publish
GitHub Actions generates a fresh brief every weekday at 8:00 AM ET using Claude Sonnet 4.6 with web search.

Requires `ANTHROPIC_API_KEY` secret in repo settings.
